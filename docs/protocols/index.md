# Protocols

## Communication & Collaboration

OnePersonLab uses standardized protocols for agent coordination.

---

## Available Protocols

### [Skills Configuration](skills.md)

Universal and discipline-specific skills setup.

### [Cross-Discipline Collaboration](cross-discipline.md)

Protocol for multi-discipline research projects.

### [PI Council](pi-council.md)

Weekly synchronization between all PIs and Lab-Director.

### [Communication Bus](communication-bus.md)

Standardized message format for agent-to-agent communication.

### [Resource Sharing](resource-sharing.md)

Shared resource management (compute, data, tools).

---

## Quick Reference

| Protocol | Purpose | Frequency |
|----------|---------|-----------|
| Skills Setup | Configure agent capabilities | Once per project |
| PI Council | Cross-discipline coordination | Weekly |
| Communication Bus | Agent messaging | Continuous |
| Resource Sharing | Allocate shared resources | As needed |
