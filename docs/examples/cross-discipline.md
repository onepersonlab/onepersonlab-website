# Cross-Discipline Project Example

## Project: AI-Driven Drug Discovery

### Overview

Cross-discipline collaboration between Chemistry, Biology, and Computer Science.

---

### Configuration

```yaml
project:
  name: "AI-Driven Drug Discovery Platform"
  type: cross-discipline
  lead_pi: CHEM-PI
  co_pi:
    - BIO-PI
    - CS-PI
  members:
    - CHEM-PD-ORG
    - BIO-PD-MOL
    - CS-PD-AI
    - CHEM-GR-03-CATAL
    - BIO-GR-01-GENO
    - CS-GR-02-CV
  timeline: 15-days
  budget: balanced
```

---

### Workflow

#### Day 0: Skills Check
- Verify universal skills for all agents
- Discuss project-specific skill needs

#### Day 1-3: Platform Setup
- **CS Team**: Build molecular generation model
- **CHEM Team**: Define target chemical space
- **BIO Team**: Identify therapeutic targets

#### Day 4-8: Candidate Generation
- Generate candidate molecules
- Evaluate synthetic feasibility
- Assess target binding predictions

#### Day 9-12: Analysis and Refinement
- Joint analysis of candidates
- Provide synthesis feedback
- Refine target selection

#### Day 13-15: Documentation
- Write project report
- Prepare paper draft

---

### Skills Used

**Universal** (All Agents):
- memory-setup, tavily-search, academic-research-hub
- self-improving, find-skills, nano-pdf

**Discipline-Specific** (Examples):
- Chemistry: cheminformatics, molecular-modeling
- Biology: genomics-tools, pathway-analysis
- CS: deep-learning-frameworks, molecular-generation

---

### Success Metrics

- Platform generates valid molecules ✓
- At least 20 candidates pass feasibility filter ✓
- Project report completed in 15 days ✓
- Paper draft ready for submission ✓
